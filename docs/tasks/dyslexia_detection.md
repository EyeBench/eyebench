# Dyslexia Detection

**Prediction target**: binary label indicating clinically diagnosed developmental dyslexia

**Task type**: classification over reader categories

## Supported datasets

- [CopCo](/datasets/copco)

Dyslexia Detection reflects EyeBench's reader-centric clinical screening objective. Trials combine naturalistic
reading scanpaths with verified diagnostic labels, challenging models to identify subtle gaze signatures associated
with dyslexia while controlling for age and language background.
